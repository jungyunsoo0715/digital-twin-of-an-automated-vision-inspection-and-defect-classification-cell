# 5주 구현 로드맵

하루 약 6시간, 매주 실행 가능한 결과물을 남기는 일정이다. 각 주의 완료 기준을 만족하지 못하면 다음 주 확장 항목보다 미완료 항목을 우선한다.

## 1주차 — OpenUSD와 셀 외형

- Isaac Sim/OpenUSD 기본 예제 확인, 에셋 후보와 라이선스 기록
- 단위·축·스케일 정리, 장비 placeholder 교체
- Reference/Sublayer 구조, 조명, 시연 카메라 구성
- 완료 기준: 주요 장비의 크기·동선이 타당한 정적 생산 셀

## 2주차 — 컨베이어와 제품 이동

- Rigidbody/Collider, 마찰, 속도, 제품 생성 간격 조정
- 감지 센서, 검사 위치 정지, 재시작 구현
- 30개 연속 이동 안정성 시험
- 완료 기준: 제품 이탈·중첩·이중 감지 없이 검사 위치 통과

## 3주차 — 검사와 로봇 분류

- 확률 기반 판정, 고정 Pick/Place, 상태 머신 연결
- PASS 연속 배출과 FAIL 불량 박스 이동
- 완료 기준: 처음부터 끝까지 무인 사이클이 반복 실행

## 4주차 — KPI와 장애

- KPI, 이벤트 로그, 경광등/설비 상태 표시
- 부품 걸림과 픽업 실패 중 최소 1개 완성, Reset 추가
- 완료 기준: 장애 재현 → 안전 정지 → 원인 표시 → 복구 시연

## 5주차 — What-if와 포트폴리오

- 0.2/0.35/0.5 m/s 동일 조건 실험, 병목과 처리량 비교
- 물리 안정성/프레임 최적화, 문서·다이어그램·코드 정리
- 2~3분 전체 시연과 30초 핵심 영상 제작
- 완료 기준: 장면, 코드, 결과 데이터, 문서, 영상으로 주장을 입증

## 주간 공통 체크

- 금요일: 기능 동결 및 회귀 시험
- 토요일: 결과 캡처, 알려진 문제 기록, 다음 주 범위 조정
- 모든 실험: 설정, seed, 코드 버전, 결과 파일을 함께 보관
